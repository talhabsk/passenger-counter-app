# Passenger Counter App

A simple browser-based application to count passengers and manage capacity limits. It is highly adaptable and can be quickly repurposed for events, stores, or any other custom tracking needs.

## Features
- Increments the count with a simple button.
- Automatically stops at a maximum limit (default: 10, easily adjustable in code).
- Highlights the counter in red when the limit is reached.
- Saves the current count to a history list without losing the total capacity.
- Resets the counter, history, and total capacity.
